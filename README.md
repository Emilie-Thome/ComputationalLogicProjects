# Implementing a propositional prover

Compilation de tous les codes : `make all_prover`

Démarrer Emacs avec prover_tests.ml : `make prover_tests`

Faire toutes les preuves : `make all_proof`

L'ensemble des preuves est :

    - add.proof
    - andcomm.proof
    - app.proof
    - appid.proof
    - bool.proof
    - comp.proof
    - contr.proof
    - curry1.proof
    - curry2.proof
    - diag.proof
    - dist.proof
    - felim.proof
    - impdm.proof
    - injl.proof
    - injr.proof
    - nnef.proof
    - nnem.proof
    - nni.proof
    - ntf.proof
    - orcomm.proof
    - orelim.proof
    - pred.proof
    - russel.proof
    - s.proof
    - tintro.proof
    - tstr.proof
    
Elles peuvent être testées, par exemple : `make add.proof`
